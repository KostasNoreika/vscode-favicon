Configure VS Code Favicon Extension
API Base URL
Set the base URL for the favicon API server. This is useful for self-hosting or testing with a local server.
HTTPS required (or http://localhost for development)
Information
Default URL: https://favicon-api.noreika.lt

Security: Only HTTPS URLs are accepted for remote servers. HTTP is allowed only for localhost/127.0.0.1 for development purposes.

Examples:
• Production: https://favicon-api.noreika.lt
• Self-hosted: https://your-server.com
• Local dev: http://localhost:8090
